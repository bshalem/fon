# fon
FON: a working set extracted from an entity linking   (EL) dataset and  augmented to support mentions in phonological representation to relieve memory-impaired named entity recognition (NER).

# What is FON?

The working set is formatted as a collection of json objects
```json
{
  "name": "John Doe",
# Download FON
'''
The single file to download the entire data is [prompt_story.json](prompt_story.json).
